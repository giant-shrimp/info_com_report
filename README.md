# 情報通信論 / Information and Communication Theory

このリポジトリは、「情報通信論（Information and Communication Theory）」で出題された課題のソースコードおよび検証データを管理するプロジェクトです。

各課題は独立したディレクトリに分割されており、それぞれの理論的背景に基づいたアルゴリズムの実装・評価・可視化をフルスクラッチで行っています。

## 📁 プロジェクト構成

リポジトリは現在、以下の2つの主要な課題ディレクトリで構成されています。

### 1. [data_compression](./data_compression) (データ圧縮・ブロック整列法 ベンチマーク)
各種データ圧縮アルゴリズム（ハフマン符号、算術符号、LZSS、LZW）に対するブロック整列法（BWT + MTF）の前処理効果を検証するプロジェクトです。

- **特徴**: 極限まで高速化されたC言語実装のBWTと、各圧縮アルゴリズムを用いた自動ベンチマークパイプライン。
- **検証項目**: 数十kB〜数MBの実テキストや境界ケース（0バイトファイル等）に対する完全なラウンドトリップテストと、圧縮率・処理時間の計測。

### 2. [error_correction](./error_correction) (畳み込み符号 誤り訂正能力評価)
1ビット入力2ビット出力の畳み込み符号において、遅延素子の個数および生成多項式の違いによる誤り訂正能力を評価するプロジェクトです。

- **特徴**: G1 (1,3)₈, G2 (5,7)₈, G3 (5,13)₈ および最適符号 G4 (15,17)₈ の4種類の符号器と、状態遷移（トレリス）を自動構築する硬判定ビタビ復号器のC言語実装。
- **検証項目**: 限界付近のブレを排除するための「低誤り数での全パターン列挙」と「高誤り数での大規模ランダム抽出」を組み合わせたハイブリッド手法による高精度な成功率曲線の導出。

## 統一されたディレクトリ設計思想

両プロジェクトともに、役割ごとに整理されたクリーンなディレクトリ構成を採用しています。

- **`c/`**: メインのアルゴリズム処理を行うC言語のソースコード群。
- **`python/`**: 評価自動化スクリプトや、グラフ生成などの補助ツール。
- **`output/`**: コンパイル済みバイナリ、中間ファイル、および最終的な成果物（CSVデータやグラフ画像）の出力先。ソースコードと成果物が混ざらないよう分離しています。

## 動作環境・必須ライブラリ

本リポジトリのプログラムをコンパイルおよび実行し、検証を行うためには以下の環境が必要です。

### 1. C言語ビルド環境
各アルゴリズムのコアとなるCプログラムのコンパイルに使用します。
- **GCC** (GNU Compiler Collection)
- **Make**

### 2. Python環境
テストの自動実行パイプライン、ベンチマーク処理、および結果のグラフ描画に使用します。
- **Python 3.x**
- **Matplotlib** (結果のグラフ生成に必要)
  - 例: `pip install matplotlib` 等でインストールしてください。
  - ※ その他の `csv`, `os`, `sys`, `subprocess` などはすべてPythonの標準ライブラリを使用しているため、追加のパッケージインストールは不要です。
